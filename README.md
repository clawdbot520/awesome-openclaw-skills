# Awesome OpenClaw Skills

A collection of custom skills for OpenClaw.

## Skills

| Skill | Description |
|-------|-------------|
| [claude-code-bridge](./claude-code-bridge/) | Dispatch coding tasks to Claude Code with Telegram notifications (free alternative to Remote Control) |
| [claude-code-memory](./claude-code-memory/) | Bidirectional memory between Claude Code TUI and OpenClaw agents via [memory-lancedb-pro](https://github.com/win4r/memory-lancedb-pro) |
| [podcast-transcriber](./podcast-transcriber/) | Automatically transcribe Podcast audio to text |
| [twitter-scraper](./twitter-scraper/) | Scrape X (Twitter) content to Markdown |

## Installation

```bash
# Clone to OpenClaw skills folder
cp -r claude-code-bridge ~/.openclaw/skills/
cp -r claude-code-memory ~/.openclaw/skills/
cp -r podcast-transcriber ~/.openclaw/skills/
cp -r twitter-scraper ~/.openclaw/skills/
```
